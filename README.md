# blackjack.sh
A simple game of blackjack written in bash.

# Dependencies

* Bash
* Gnu Core Utilities 

# Installation

`git clone https://github.com/EmacsPenguin/blackjack.sh.git`

`cd blackjack.sh`

`chmod +x blackjack.sh`

`./blackjack.sh`

Move to your `$PATH` of you want it to be executable from anywhere in your terminal.
